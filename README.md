# whereQR
Description : Web service for stickers for dealing with loss

## 1. 실행방법

#### 서버 실행방법
###### backend
- manage.py와 동일한 위치에서 진행
``` c
python manage.py runserver
```
###### frontend
- src와 동일한 위치에서 진행
 ``` c
run start
```

###### 데이터베이스 모델 저장 및 적용
- manage.py와 동일한 위치에서 진행
``` c
python manage.py makemigrations
python manage.py migrate
```
